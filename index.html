Breakout Game

For create this App, I using JavaScript(ECMA-6), HTML-5(Canvas API) and Guide from MDN Web Docs.


1. Crate to Ball, Paddle and Bricks props.
2. Draw Ball, Paddle, and Bricks on Canvas.
3. Using Backtick for Draw Score.
4. Using For Loop, for find position every brick.
5. Using forEach Loop:
  a) For Draw Bricks on Canvas.
  b) For Create Bricks Collision with Ball
  c) For showAllBricks
6. Using If Statement:
  a) In Wall Detection
  b) In Bricks Collision with Ball
  c) In Win/Lose state
  d) In Key Events
